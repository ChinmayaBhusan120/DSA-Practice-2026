# 🌳 Binary Tree Pattern

## 🧠 Overview

A **Binary Tree** is a hierarchical data structure where each node has at most two children:

- Left child
- Right child

Binary Trees are one of the most important topics in coding interviews and are widely used in:

- Search algorithms
- Hierarchical data representation
- Expression trees
- Database indexing
- File systems
